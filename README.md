💰 Interest Calculator

A simple and interactive web application that helps users calculate Simple Interest and Compound Interest quickly and accurately. 

Built using HTML, CSS, and Vanilla JavaScript, this project is perfect for both learning and practical use!

✨ Features

🔹 User-Friendly Interface

🔹 Instant Calculations for both Simple and Compound Interest

🔹 Real-Time Input Validation

🔹 Responsive Design – works on desktops, tablets, and mobiles

🔹 100% built with Vanilla JavaScript (no external frameworks or libraries)

🔢 What It Calculates

🔸 Simple Interest

Formula used:

     SI = (Principal × Rate × Time) / 100
     
🔸 Compound Interest

Formula used:

     CI = Principal × (1 + Rate/100)^Time – Principal
     
🛠️ Tech Stack

HTML5 – Structure

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Functionality and interactivity
